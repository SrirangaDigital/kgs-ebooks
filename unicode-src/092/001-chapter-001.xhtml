1.ಜಗತ್ಕಾರಣನಾದ ಹರಿಗೆ ನಮಸ್ಕಾರ ಜನವರಿ 1.

॥ಓಂ ನೈಷ್ಕರ್ಮ್ಯಸಿದ್ಧಿಲೇಖಕಾಯ ಶ್ರೀಸುರೇಶ್ವರಾಯ ನಮಃ॥

“ನೈಷ್ಕರ್ಮ್ಯಸಿದ್ಧಿ”ಕರ್ತಾರಂ “ವಾರ್ತಿಕದ್ವಯ”ಲೇಖಕಮ್।
ಶಂಕರಾಚಾರ್ಯಸಚ್ಛಿಷ್ಯಂ ಸುರೇಶ್ವರಮಹಂ ಭಜೇ॥

ಖಾನಿಲಾಗ್ನ್ಯಬ್ಧರಿತ್ರ್ಯಂತಂ ಸ್ರಕ್ಫಣೀವೋದ್ಗತಂ ಯತಃ।
ಧ್ವಾಂತಚ್ಛಿದೇ ನಮಸ್ತಸ್ಮೈ ಹರಯೇ ಬುದ್ಧಿಸಾಕ್ಷಿಣೇ॥

ನೈಷ್ಕರ್ಮ್ಯಸಿದ್ಧಿಃ ೧-೧

ಅನ್ವಯಾರ್ಥ : ಖ ಅನಿಲ ಅಗ್ನಿ ಅಪ್ ಧರಿತ್ರೀ ಅಂತಂ = ಆಕಾಶ ವಾಯು ಅಗ್ನಿ ಅಪ್ಪು ಪೃಥಿವಿಯವರೆಗಿರುವ ಈ ಜಗತ್ತು , ಯತಃ = ಯಾವನಿಂದ, ಸ್ರಕ್ ಫಣೀ ಇವ = ಹೂಮಾಲೆಯಿಂದ ಸರ್ಪವು ಹುಟ್ಟುವಂತೆ, ಉದ್ಗತಂ = ಹುಟ್ಟಿರುತ್ತದೆಯೋ, ಧ್ವಾಂತಚ್ಛಿದೇ = ಅಜ್ಞಾನವನ್ನು ನಾಶಮಾಡುವಂಥ, ಬುದ್ಧಿಸಾಕ್ಷಿಣೇ = ನಮ್ಮ ಬುದ್ಧಿಗೆ ಸಾಕ್ಷಿಯಾಗಿರುವಂಥ, ತಸ್ಮೈ = ಆ, ಹರಯೇ = ಮಹಾವಿಷ್ಣುವಿಗೆ, ನಮಃ = ನಮಸ್ಕಾರವು.

ಭಾವಾರ್ಥ : ಹೂಮಾಲೆಯಿಂದ ಸರ್ಪವು ಹುಟ್ಟಿರುವಂತೆ, ಈ ಆಕಾಶಾದಿ ಭೂಮ್ಯಂತವಾಗಿರುವ ವಿಶ್ವವು ಯಾವ ಪರಮಾತ್ಮನಿಂದ ಹುಟ್ಟಿರುತ್ತದೆಯೋ, ಅವಿದ್ಯೆಯನ್ನು ನಾಶಮಾಡುವಂಥ, ಬುದ್ಧಿಗೆ ಸಾಕ್ಷಿಯಾಗಿರುವ ಆ ಮಹಾವಿಷ್ಣುವಿಗೆ ನಮಸ್ಕಾರ.

ವಿವರಣೆ : ಮಬ್ಬುಗತ್ತಲೆಯಲ್ಲಿ ಹೂಮಾಲೆಯೆಂದು ತಿಳಿಯದಿರುವಾಗ ಅದನ್ನೇ ಸರ್ಪವೆಂದು ತಿಳಿಯುವಂತೆ, ಪರಮಾತ್ಮನ ನಿಜಸ್ವರೂಪವನ್ನು ತಿಳಿಯದೇ, ಅವನನ್ನೇ ಜಗತ್ತೆಂದು ಅಜ್ಞಾನಿಗಳು ಭ್ರಮಿಸಿರುತ್ತಾರೆ. ನಿಜವಾಗಿ ನೋಡಿದರೆ ಈ ಪ್ರಪಂಚವು ಪರಬ್ರಹ್ಮವೇ.

ಪರಮಾತ್ಮನೆಂದರೆ ದೂರದಲ್ಲಿರುವ ದೇವನಲ್ಲ. ಮತ್ತೇನೆಂದರೆ ನಮ್ಮಗಳ ಬುದ್ಧಿಗಳಿಗೆ ಸಾಕ್ಷಿಯಾಗಿರುವ ಪ್ರತ್ಯಗಾತ್ಮನೇ ದೇವನು. ಇವನು ನಮ್ಮಗಳ ಅಜ್ಞಾನವನ್ನು ನಾಶಮಾಡುತ್ತಾನೆ. ಇವನಿಗೇ ಹರಿ, ವಿಷ್ಣು—ಎಂದು ಹೆಸರು. ಇವನಿಗೆ ನಮಸ್ಕಾರ.

॥ಓಂ ಶಾಂತಿಃ ಶಾಂತಿಃ ಶಾಂತಿಃ॥

2. ಜ್ಞಾನಿಗೆ ಶೋಕಮೋಹಗಳಿಲ್ಲ ಜನವರಿ 2.

॥ಓಂ ಸಂಸಾರಭಯರಹಿತಾಯ ಶ್ರೀಸುರೇಶ್ವರಾಯ ನಮಃ॥

ಏಕತ್ವದರ್ಶನಾದೇವ ಶೋಕಮೋಹವಿವರ್ಜಿತಮ್।
ಸಂಸಾರಬಂಧನಾತೀತಂ ಸುರೇಶ್ವರಮಹಂ ಭಜೇ॥

ತತ್ರಕೋ ಮೋಹಃ ಕಃ ಶೋಕಃ ಇತಿ ಮಂತ್ರೋಽಪಿ ನೋ ಜಗೌ।
ಸಂಸಾರಾಸಂಭವಂ ಸಾಕ್ಷಾತ್ ಏಕತ್ವಮನುಪಶ್ಯತಃ॥

ಬೃಹದಾರಣ್ಯಕಭಾಷ್ಯವಾರ್ತಿಕಂ ೧-೪-೮೧೫

ಅನ್ವಯಾರ್ಥ : ‘ತತ್ರ ಕೋ ಮೋಹಃ ಕಃ ಶೋಕಃ’ = ಅವನಿಗೆ ಯಾವ ಮೋಹವು? ಯಾವ ಶೋಕವಿದೆ?, ಇತಿ ಮಂತ್ರಃ ಅಪಿ = ಎಂಬ ಈ ಮಂತ್ರವೂ ಕೂಡ, ಸಾಕ್ಷಾತ್ = ನೇರಾಗಿ, ಏಕತ್ವಮ್ = ಏಕತ್ವವನ್ನು, ಅನುಪಶ್ಯತಃ = ಕಾಣುವವನಿಗೆ, ಸಂಸಾರಾಸಂಭವಂ = ಸಂಸಾರವು ಇಲ್ಲ ಎನ್ನುವುದನ್ನು, ನಃ = ನಮಗೆ, ಜಗೌ = ಹೇಳುತ್ತಿದೆ.

ಭಾವಾರ್ಥ : ‘ತತ್ರ ಕೋ ಮೋಹಃ ಕಃ ಶೋಕಃ’ ಎಂಬೀ ಈ ಈಶಾವಾಸ್ಯ ಉಪನಿಷತ್ ಮಂತ್ರವು ಆತ್ಮೈಕತ್ವವನ್ನು ಕಂಡವನಿಗೆ ಸುಖದುಃಖರೂಪವಾದ ಸಂಸಾರವಿಲ್ಲವೆಂದು ಸ್ಪಷ್ಟವಾಗಿ ತಿಳಿಸುತ್ತಿದೆ.

ವಿವರಣೆ : ‘ಏಕತ್ವವನ್ನು ಅರಿತವನಿಗೆ ಶೋಕವಾಗಲೀ ಮೋಹವಾಗಲೀ ಇರುವುದಿಲ್ಲ’ ಎಂದು ಈಶಾವಾಸ್ಯ ಮಂತ್ರವು ಹೇಳುತ್ತಿದೆ. ಇದು ಅನುಭವಕ್ಕೂ ಸಂಮತವಾಗಿರುತ್ತದೆ. ಮೋಹವೆಂದರೆ ಅನಾತ್ಮವಸ್ತುಗಳಲ್ಲಿ ನಾನು ಎಂಬ ಭ್ರಾಂತಿಜ್ಞಾನವು. ಶೋಕವೆಂದರೆ ಆ ವಸ್ತುಗಳ ನಾಶನಿಮಿತ್ತವಾದ ದುಃಖವು. ಅಜ್ಞಾನದಿಂದಲೇ ಶೋಕಮೋಹಗಳು ತೋರುತ್ತಿವೆ. ಜ್ಞಾನವುಂಟಾಯಿತೆಂದರೆ ಅಜ್ಞಾನವು ನಾಶವಾಗುತ್ತದೆ. ಈ ಅಜ್ಞಾನವು ತೊಲಗಿದಮೇಲೂ ಅದರ ಕಾರ್ಯವಾದ ಸಂಸಾರಾನುಭವವು ಬ್ರಹ್ಮಜ್ಞಾನಿಗೆ ಹೇಗೆ ತಾನೆ ಇದ್ದೀತು? ಖಂಡಿತಾ ಇರುವುದಿಲ್ಲ. ಏಕೆಂದರೆ ಬ್ರಹ್ಮಾನುಭವವನ್ನುಳ್ಳವನಿಗೆ ದ್ವೈತವಸ್ತುಗಳಲ್ಲಿ ಸತ್ಯತ್ವಭ್ರಮೆಯೇ ಇರುವುದಿಲ್ಲ. ಹೀಗಾಗಿ ಬ್ರಹ್ಮಜ್ಞಾನಿಗೆ ಶೋಕಮೋಹಗಳಿರುವುದಿಲ್ಲ.

॥ಓಂ ಶಾಂತಿಃ ಶಾಂತಿಃ ಶಾಂತಿಃ॥

3.ಜ್ಞಾನೋಪದೇಶವೇ ಗುರಿ ಜನವರಿ 2.

॥ಓಂ ಸಂಸಾರರೋಗವರ್ಜಿತಾಯ ಶ್ರೀಸುರೇಶ್ವರಾಯ ನಮಃ॥

ಸಂಸಾರಕಾರಣೀಭೂತಭ್ರಾಂತಿಜ್ಞಾನನಿವಾರಕಮ್।
ಜ್ಞಾನನಿಷ್ಠಾಪರಂ ಶಾಂತಂ ಸುರೇಶ್ವರಮಹಂ ಭಜೇ॥

ಸಂಸಾರಕಾರಣಾವಿದ್ಯಾಧ್ವಂಸಕೃತ್ ಜ್ಞಾನಲಬ್ಧಯೇ।
ಪ್ರಾರಬ್ಧೇಯಂ ಪ್ರಯತ್ನೇನ ವೇದಾಂತೋಪನಿಷತ್ ಪರಾ॥

ಸಂಬಂಧವಾರ್ತಿಕಂ ೧೭

ಅನ್ವಯಾರ್ಥ : ಸಂಸಾರ ಕಾರಣ ಅವಿದ್ಯಾ ಧ್ವಂಸಕೃತ್ ಜ್ಞಾನಲಬ್ಧಯೇ = ಸಂಸಾರಕ್ಕೆ ಕಾರಣವಾದ ಅವಿದ್ಯೆಯನ್ನು ನಾಶಮಾಡುವಂಥ ಆತ್ಮಜ್ಞಾನವನ್ನು ತಿಳಿಸುವುದಕ್ಕಾಗಿ, ಇಯಂ = ಈ, ಪರಾ = ಶ್ರೇಷ್ಠವಾದ, ವೇದಾಂತೋಪನಿಷತ್ = ವೇದಾಂತವೆಂಬ ಉಪನಿಷತ್ತು, ಪ್ರಯತ್ನೇನ = ಪ್ರಯತ್ನಪೂರ್ವಕವಾಗಿ, ಪ್ರಾರಬ್ಧಾ = ಪ್ರಾರಂಭಿಸಲ್ಪಟ್ಟಿರುತ್ತದೆ.

ಭಾವಾರ್ಥ : ಸಂಸಾರಕ್ಕೆ ಕಾರಣವಾದದ್ದು ಅವಿದ್ಯೆಯು, ಇದನ್ನು ನಾಶಮಾಡುವಂಥದ್ದು ಆತ್ಮಜ್ಞಾನವು. ಇದನ್ನು ಉಪದೇಶಿಸುವುದಕ್ಕಾಗಿ ಈ ಶ್ರೇಷ್ಠವಾದ ಬೃಹದಾರಣ್ಯಕೋಪನಿಷತ್ತು ಹೊರಟಿದೆ. ಇದನ್ನು ನಾವು ಪ್ರಯತ್ನಪಟ್ಟು ವ್ಯಾಖ್ಯಾನಮಾಡುತ್ತಿದ್ದೇವೆ.

ವಿವರಣೆ : ಜನ್ಮಮರಣರೂಪವಾದ ಈ ಸಂಸಾರಕ್ಕೆ ಅಜ್ಞಾನವೇ ಕಾರಣ. ತನ್ನ ನಿಜಸ್ವರೂಪವನ್ನು ಸದ್ಗುರುಗಳಿಂದ ಸರಿಯಾಗಿ ತಿಳಿಯದಿರುವುದೇ ಅಜ್ಞಾನ. ಇದರಿಂದ ದೇಹಾದಿಗಳಲ್ಲಿ ಅಧ್ಯಾಸ. ಅದರಿಂದ ಕಾಮಕರ್ಮಸುಖದುಃಖಗಳು. ಇದೆಲ್ಲಕ್ಕೂ ಅಜ್ಞಾನವೇ ಕಾರಣ. ಅಜ್ಞಾನವನ್ನು ಆತ್ಮಜ್ಞಾನದಿಂದ ಮಾತ್ರವೇ ಕಳೆದುಕೊಳ್ಳಬೇಕು. ಸಂಸಾರವು ಚಕ್ರದಂತೆ ಇರುತ್ತದೆ. ಜನ್ಮಮರಣಗಳು, ಸುಖದುಃಖಗಳು, ರಾಗದ್ವೇಷಗಳು, ಲಾಭನಷ್ಟಗಳು—ಇದೆಲ್ಲವೂ ಸಂಸಾರವೇ. ಇದಕ್ಕೇ ಅನರ್ಥಪರಂಪರೆ—ಎಂದೂ ಹೆಸರು. ಇದಕ್ಕೆ ಅಜ್ಞಾನವೇ ಹೇತು. ಅಜ್ಞಾನನಿವೃತ್ತಿಗೆ ಆತ್ಮಜ್ಞಾನವೇ ಬೇಕು. ಇದನ್ನು ಮುಮುಕ್ಷುಗಳು ಅರ್ಥಮಾಡಿಕೊಂಡು ಕೃತಾರ್ಥರಾಗಬೇಕು.

॥ಓಂ ಶಾಂತಿಃ ಶಾಂತಿಃ ಶಾಂತಿಃ॥

4.ತನ್ನೊಳಗೇ ಬ್ರಹ್ಮವಿದೆ ಜನವರಿ 2.

॥ಓಂ ಸತ್ಯಜ್ಞಾನಸ್ವರೂಪಾಯ ಶ್ರೀಸುರೇಶ್ವರಾಯ ನಮಃ॥

ಸರ್ವದೋಷವಿನಿರ್ಮುಕ್ತಂ ಬ್ರಹ್ಮ ಪ್ರಾಪ್ನೋತಿ ವಿದ್ಯಯಾ।
ನಿರ್ಧೂತಾಶೇಷಸಂಸಾರಂ ಸುರೇಶ್ವರಮಹಂ ಭಜೇ ॥

ನಿರ್ಧೂತಾಶೇಷಸಂಸಾರಃ ಸತ್ಯಮಿತ್ಯಾದಿನೋದಿತಃ।
ವ್ಯುತ್ಥಾಪ್ಯಾಸತ್ಯಾಜ್ಞಾನಾದೇಃ ಬುದ್ಧೌ ಚಾತ್ಮನಿ ದರ್ಶಿತಃ॥

ತೈತ್ತಿರೀಯೋಪನಿಷದ್ಭಾಷ್ಯವಾರ್ತಿಕಂ, ಬ್ರಹ್ಮಾನಂದವಲ್ಲೀ ೫೨೨

ಅನ್ವಯಾರ್ಥ :ಸತ್ಯಮ್ ಇತ್ಯಾದಿನಾ ಉದಿತಃ = ಸತ್ಯಂ ಜ್ಞಾನಂ ಅನಂತಂ ಬ್ರಹ್ಮ—ಮುಂತಾದ ಶ್ರುತಿವಾಕ್ಯದಿಂದ ತಿಳಿಸಲ್ಪಟ್ಟ ಆತ್ಮನು, ನಿರ್ಧೂತ ಅಶೇಷಸಂಸಾರಃ = ಯಾವ ಸಂಸಾರದ ಸಂಬಂಧವೂ ಇಲ್ಲದವನು. ಅಸತ್ಯ ಅಜ್ಞಾನಾದೇಃ = ಅಸತ್ಯ ಹಾಗೂ ಅಜ್ಞಾನಗಳಿಂದ, ವ್ಯುತ್ಥಾಪ್ಯ = ಮೇಲಕ್ಕೆ ಏಳಿಸಲ್ಪಟ್ಟು, ಬುದ್ಧೌ = ಬುದ್ಧಿಯಲ್ಲಿ, ಚ = ಮತ್ತು , ಆತ್ಮನಿ = ತನ್ನೊಳಗೇ, ದರ್ಶಿತಃ = ತೋರಿಸಲ್ಪಟ್ಟಿರುತ್ತಾನೆ.

ಭಾವಾರ್ಥ : ಶ್ರುತಿಯು ಆತ್ಮನನ್ನು ಸತ್ಯಂ ಜ್ಞಾನಂ ಅನಂತಂ ಎಂದು ಹೇಳಿ, ಆತ್ಮನಲ್ಲಿ ಸಂಸಾರದ ಸಂಬಂಧವು ಇಲ್ಲ ಎಂದು ತಿಳಿಸುತ್ತಿದೆ. ಈ ಆತ್ಮನಲ್ಲಿ ಅಸತ್ಯ ಅಜ್ಞಾನ ಮುಂತಾದ ದೋಷಗಳಾವುವೂ ಇಲ್ಲ ಎಂದು ಹೇಳುತ್ತಾ ನಮ್ಮೊಳಗೇ ಈ ಆತ್ಮನಿದ್ದಾನೆ ಎಂದು ಈ ಶ್ರುತಿಯು ಹೇಳುತ್ತಿದೆ.

ವಿವರಣೆ : ಉಪನಿಷತ್ತುಗಳ ಸಂದೇಶದಂತೆ ಬ್ರಹ್ಮವೇ ಆತ್ಮ, ಆತ್ಮನೇ ಬ್ರಹ್ಮ. ಈ ಆತ್ಮನು ಸಕಲಸಂಸಾರದೋಷರಹಿತನಾಗಿರುತ್ತಾನೆ. ಆತ್ಮನಲ್ಲಿ ಅವಿದ್ಯೆಯಿಂದಲೇ ಅಸತ್ಯತ್ವ, ಅಜ್ಞಾನ ಮುಂತಾದ ದೋಷಗಳು ತೋರುತ್ತಿವೆ. ಶ್ರುತಿಯು ಈ ದೋಷಗಳನ್ನೆಲ್ಲ ಕಿತ್ತುಹಾಕಿ, ಈ ಆತ್ಮನು ನಮ್ಮೊಳಗೇ ಇದ್ದಾನೆ ಎಂದು ತಿಳಿಸುತ್ತಿದೆ. ಔಪನಿಷದ ಆತ್ಮನು ನಿರುಪಾಧಿಕನಾಗಿಯೂ ಪರಿಪೂರ್ಣನಾಗಿಯೂ ಇದ್ದು , ಸರ್ವರಿಗೂ ಪ್ರತ್ಯಗಾತ್ಮನೇ ಆಗಿರುತ್ತಾನೆ. ಈ ಪ್ರತ್ಯಗಾತ್ಮನನ್ನೇ ಸಾಧಕರು ತಮ್ಮ ಸ್ವರೂಪವೆಂದು ತಿಳಿಯಬೇಕು.

॥ಓಂ ಶಾಂತಿಃ ಶಾಂತಿಃ ಶಾಂತಿಃ॥

5.ಜ್ಞಾನಿಗಳ ನಿಶ್ಚಯಜ್ಞಾನ ಜನವರಿ 2.

॥ಓಂ ಸ್ವಪ್ರಕಾಶಾಯ ಶ್ರೀಸುರೇಶ್ವರಾಯ ನಮಃ॥

ಆತ್ಮನೈವ ಯಥಾ ಸ್ವಪ್ನಃ ತಥಾ ಜಾಗ್ರದ್ ವಿಭಾಸ್ಯತೇ।
ಸ್ವಯಂಪ್ರಕಾಶರೂಪಂ ಶ್ರೀಸುರೇಶ್ವರಮಹಂ ಭಜೇ॥

ಸ್ವಪ್ನೇ ಪ್ರಕಾಶೋ ಭಾವಾನಾಂ ಸ್ವಪ್ರಕಾಶಾನ್ನ ಹೀತರಃ।
ಜಾಗ್ರತ್ಯಪಿ ತಥೈವೇತಿ ನಿಶ್ಚಿನ್ವಂತಿ ವಿಪಶ್ಚಿತಃ॥

ಮಾನಸೋಲ್ಲಾಸ ೧-೧೧

ಅನ್ವಯಾರ್ಥ :ಸ್ವಪ್ನೇ= ಕನಸಿನಲ್ಲಿ , ಭಾವಾನಾಂ = ಪದಾರ್ಥಗಳ, ಪ್ರಕಾಶಃ = ತೋರುವಿಕೆಯು, ಸ್ವಪ್ರಕಾಶಾತ್ = ತನ್ನ ತೋರುವಿಕೆಗಿಂತ, ಇತರಃ ನ ಭವತಿ = ಭಿನ್ನವಾಗಿರಲಾರದೋ, ತಥಾ ಏವ = ಅದೇ ರೀತಿಯಲ್ಲಿ, ಜಾಗ್ರತಿ ಅಪಿ = ಎಚ್ಚರದಲ್ಲಿಯೂ ಕೂಡ, ಇತಿ = ಎಂದು, ವಿಪಶ್ಚಿತಃ = ಜ್ಞಾನಿಗಳು, ನಿಶ್ಚಿನ್ವಂತಿ = ನಿಶ್ಚಯಿಸುತ್ತಾರೆ.

ಭಾವಾರ್ಥ : ಕನಸಿನಲ್ಲಿ ತೋರುತ್ತಿದ್ದ ಸಕಲವಸ್ತುಗಳಿಗೂ ತನ್ನ ಇರುವಿಕೆಯೇ ಆಸ್ಪದವಾಗಿತ್ತಲ್ಲವೆ? ಹಾಗೆಯೇ ಈ ಎಚ್ಚರದಲ್ಲಿಯೂ ಎಂದು ಜ್ಞಾನಿಗಳು ತಿಳಿಯುತ್ತಾರೆ.

ವಿವರಣೆ : ಕನಸಿನಲ್ಲಿ ಏನೇ ತೋರಿದರೂ, ಎಷ್ಟೇ ತೋರಿದರೂ, ಅದೆಲ್ಲಕ್ಕೂ ತನ್ನಿಂದಲೇ ಬೆಲೆಯು ಬಂದಿದೆಯಲ್ಲವೆ? ತನ್ನ ಇರುವಿಕೆಯಿಂದಲೇ, ತನ್ನ ಪ್ರಕಾಶದಿಂದಲೇ ಆ ಎಲ್ಲಾ ವಸ್ತುಗಳಿಗೂ ಇರುವಿಕೆಯು ಬಂದಿರುತ್ತದೆಯಲ್ಲವೆ? ಹಾಗೆಯೇ ಎಚ್ಚರದ ಈ ದ್ವೈತವಸ್ತುಗಳೆಲ್ಲಕ್ಕೂ ತಾನೇ ಮೂಲ ಆಸ್ಪದನಾಗಿದ್ದೇನೆ, ತನಗಿಂತ ಭಿನ್ನವಾಗಿ ಈ ಎಚ್ಚರವೆಂಬುದು ಇಲ್ಲವೇ ಇಲ್ಲ ಎಂಬುದು ಬ್ರಹ್ಮಜ್ಞಾನಿಗಳ ನಿಶ್ಚಯಜ್ಞಾನವು. ಕನಸಿನಂತೆಯೇ ಎಚ್ಚರವೂ ಮಿಥ್ಯೆಯೇ ಆಗಿರುತ್ತದೆ. ಕನಸೂ, ತೋರುವಾಗ ಸತ್ಯ, ಎಚ್ಚರವಾದಕೂಡಲೇ ಮಿಥ್ಯಾ; ಹೀಗೆಯೇ ಈ ಎಚ್ಚರವೂ ಹುಸಿತೋರಿಕೆಯೇ ಆಗಿರುತ್ತದೆ! ಹೀಗಾಗಿ ಆತ್ಮನೇ ಎಚ್ಚರವಾಗಿಯೂ ತೋರುತ್ತಿದ್ದಾನೆ ಎಂದು ಸಾಧಕರು ತಿಳಿಯಬೇಕು.

॥ಓಂ ಶಾಂತಿಃ ಶಾಂತಿಃ ಶಾಂತಿಃ॥

6.ಜ್ಞಾನದಿಂದ ಮುಕ್ತಿ ಜನವರಿ 2.

॥ಓಂ ಪ್ರತ್ಯಗರ್ಥನಿಷ್ಠಾಯ ಶ್ರೀಸುರೇಶ್ವರಾಯ ನಮಃ॥

ಪುರುಷಾರ್ಥಸತತ್ತ್ವಜ್ಞಂ ಪುರುಷಾರ್ಥೋಪದೇಶಕಮ್।
ಪುರುಷಾರ್ಥಸ್ವರೂಪಂ ಶ್ರೀಸುರೇಶ್ವರಮಹಂ ಭಜೇ॥

ನ ಹಿ ಪ್ರತೀಚಿ ವಿಜ್ಞಾತೇ ಕಶ್ಚಿದಪ್ಯವಶಿಷ್ಯತೇ।
ಜ್ಞೇಯೋಽಥರ್ಃ ಪುರುಷಾರ್ಥೋ ವಾ ಪದನೀಯಮತೋ ಭವೇತ್॥

ಬೃಹದಾರಣ್ಯಕಭಾಷ್ಯವಾರ್ತಿಕಂ ೧-೪-೯೬೬

ಅನ್ವಯಾರ್ಥ :ಪ್ರತೀಚಿ ವಿಜ್ಞಾತೇ = ಪ್ರತ್ಯಗಾತ್ಮನನ್ನು ತಿಳಿದ ಮೇಲೆ, ಕಶ್ಚಿತ್ ಅಪಿ = ಇನ್ನು ಬೇರೆ ಏನೂ, ಜ್ಞೇಯಃ ಅರ್ಥಃ ನ ಅವಶಿಷ್ಯತೇ = ತಿಳಿಯಬೇಕಾದ ವಸ್ತುವು ಉಳಿದಿರುವುದಿಲ್ಲ. ವಾ = ಅಥವಾ, ಪುರುಷಾರ್ಥಃ = ಬೇರೆಯಾಗಿ ಪುರುಷಾರ್ಥವೂ, ನ ಅವಶಿಷ್ಯತೇ = ಉಳಿದಿರುವುದಿಲ್ಲ. ಅತಃ = ಆದ್ದರಿಂದ, ಪದನೀಯಂ ಭವೇತ್ = ವಿವೇಕಿಯಾದ ಸಾಧಕನು ಪ್ರತ್ಯಗಾತ್ಮನನ್ನೇ ತಿಳಿಯಬೇಕು, ಅವನನ್ನೇ ಪಡೆಯಬೇಕು.

ಭಾವಾರ್ಥ : ನಮ್ಮ ಪ್ರತ್ಯಗಾತ್ಮನನ್ನು ತಿಳಿದಮೇಲೆ ಇನ್ನು ಬೇರೆಯಾಗಿ ತಿಳಿಯಬೇಕಾದದ್ದಾಗಲೀ ಪಡೆಯಬೇಕಾದದ್ದಾಗಲೀ ಏನೇನೂ ಉಳಿದಿರುವುದಿಲ್ಲ. ಆದ್ದರಿಂದ ವಿವೇಕಿಗಳು ಈ ಪ್ರತ್ಯಗಾತ್ಮನನ್ನೇ ತಿಳಿಯಲು ಪ್ರಯತ್ನಿಸಬೇಕು.

ವಿವರಣೆ : ಪ್ರತ್ಯಗಾತ್ಮನೆಂದರೆ ಬ್ರಹ್ಮಸ್ವರೂಪನೇ. ಪ್ರತ್ಯಗಾತ್ಮನೇ ಇಡೀ ವಿಶ್ವಕ್ಕೆ ಆಸ್ಪದನೂ, ಕಾರಣವೂ ಆಗಿರುತ್ತಾನೆ. ಆತ್ಮೈವೇದಂಸರ್ವಂ—ಎನ್ನುವಂತೆ ಇವನೇ ಎಲ್ಲವೂ ಆಗಿರುತ್ತಾನೆ. ಇವನೊಬ್ಬನನ್ನು ತಾನೆಂದು ಅನುಭವದಿಂದ ಸರಿಯಾಗಿ ತಿಳಿದರೆ ಬೇರೆ ತಿಳಿಯಬೇಕಾದದ್ದು ಏನೇನೂ ಉಳಿದಿರುವುದಿಲ್ಲ. ಆದ್ದರಿಂದಲೇ ಶ್ರುತಿಗಳಲ್ಲಿ ಆತ್ಮಾ ವೈ ದ್ರಷ್ಟವ್ಯಃ—ಆತ್ಮನನ್ನೇ ತಿಳಿದುಕೊಳ್ಳಬೇಕು ಎಂದು ಹೇಳಿರುತ್ತದೆ. ಯಜ್ಜ್ಞಾತ್ವಾ ನೇಹಭೂಯೋಽನ್ಯತ್ ಜ್ಞಾತವ್ಯಮ್ ಅವಶಿಷ್ಯತೇ ಎಂದು ಭಗವದ್ಗೀತೆಯಲ್ಲಿಯೂ ಇದನ್ನೇ ಹೇಳಿರುತ್ತದೆ. ಇದಕ್ಕೇ ಸರ್ವಾತ್ಮದರ್ಶನ ಎಂದೂ ಹೆಸರು.

॥ಓಂ ಶಾಂತಿಃ ಶಾಂತಿಃ ಶಾಂತಿಃ॥
